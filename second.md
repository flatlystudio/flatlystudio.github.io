## **a. Sistem Persamaan:**
- x - 2y + 3z = 2
- 2x - 3y - 4z = -5
- 3x + 4y + 5z = 12

### **Penyelesaian:**

**Langkah 1:** Eliminasi x dari persamaan (1) dan (2)

Kalikan persamaan (1) dengan 2:
```
2x - 4y + 6z = 4
```

Kurangkan dengan persamaan (2):
```
  2x - 4y + 6z = 4
-(2x - 3y - 4z = -5)
────────────────────
  0 - y + 10z = 9
```
Hasil: **-y + 10z = 9** atau **y = 10z - 9** ... (4)

**Langkah 2:** Eliminasi x dari persamaan (1) dan (3)

Kalikan persamaan (1) dengan 3:
```
3x - 6y + 9z = 6
```

Kurangkan dengan persamaan (3):
```
  3x - 6y + 9z = 6
-(3x + 4y + 5z = 12)
────────────────────
  0 - 10y + 4z = -6
```
Hasil: **-10y + 4z = -6** atau **10y - 4z = 6** ... (5)

**Langkah 3:** Substitusi y = 10z - 9 ke persamaan (5)
```
10(10z - 9) - 4z = 6
100z - 90 - 4z = 6
96z = 96
z = 1
```

**Langkah 4:** Cari y
```
y = 10(1) - 9 = 1
```

**Langkah 5:** Substitusi y = 1 dan z = 1 ke persamaan (1)
```
x - 2(1) + 3(1) = 2
x - 2 + 3 = 2
x = 1
```

### **Jawaban: x = 1, y = 1, z = 1**

## **b. Sistem Persamaan:**
- 2x - 4y + z = -10
- 3x + y + 2z = 60
- -2x + y - z = -20

### **Penyelesaian:**

**Langkah 1:** Eliminasi x dari persamaan (1) dan (3)

Tambahkan persamaan (1) dan (3):
```
  2x - 4y + z = -10
+(-2x + y - z = -20)
────────────────────
  0 - 3y + 0 = -30
```
```
-3y = -30
y = 10
```

**Langkah 2:** Eliminasi x dari persamaan (1) dan (2)

Kalikan persamaan (1) dengan 3 dan persamaan (2) dengan 2:
```
6x - 12y + 3z = -30
6x + 2y + 4z = 120
```

Kurangkan:
```
  6x - 12y + 3z = -30
-(6x + 2y + 4z = 120)
──────────────────────
  0 - 14y - z = -150
```
Hasil: **-14y - z = -150** ... (4)

**Langkah 3:** Substitusi y = 10 ke persamaan (4)
```
-14(10) - z = -150
-140 - z = -150
z = 10
```

**Langkah 4:** Substitusi y = 10 dan z = 10 ke persamaan (1)
```
2x - 4(10) + 10 = -10
2x - 40 + 10 = -10
2x = 20
x = 10
```

### **Jawaban: x = 10, y = 10, z = 10**

## **c. Sistem Persamaan:**
- x - 4y + z = 6
- 2x + y - 2z = -1
- 3x + 2y - 2z = -2

### **Penyelesaian:**

**Langkah 1:** Eliminasi z dari persamaan (2) dan (3)
```
  2x + y - 2z = -1
-(3x + 2y - 2z = -2)
────────────────────
  -x - y + 0 = 1
```
Hasil: **-x - y = 1** atau **x + y = -1** ... (4)

**Langkah 2:** Eliminasi z dari persamaan (1) dan (2)

Kalikan persamaan (1) dengan 2:
```
2x - 8y + 2z = 12
```

Tambahkan dengan persamaan (2):
```
  2x - 8y + 2z = 12
+(2x + y - 2z = -1)
───────────────────
  4x - 7y + 0 = 11
```
Hasil: **4x - 7y = 11** ... (5)

**Langkah 3:** Dari persamaan (4): x = -1 - y

Substitusi ke persamaan (5):
```
4(-1 - y) - 7y = 11
-4 - 4y - 7y = 11
-11y = 15
y = -15/11
```

**Langkah 4:** Cari x
```
x = -1 - (-15/11) = -1 + 15/11 = -11/11 + 15/11 = 4/11
```

**Langkah 5:** Substitusi x = 4/11 dan y = -15/11 ke persamaan (1)
```
4/11 - 4(-15/11) + z = 6
4/11 + 60/11 + z = 6
64/11 + z = 6
z = 6 - 64/11 = 66/11 - 64/11 = 2/11
```

### **Jawaban: x = 4/11, y = -15/11, z = 2/11**

## **d. Sistem Persamaan:**
- x + y - z = 13
- x + z = 11
- 2x - y = 16

### **Penyelesaian:**

Dari persamaan (2): **z = 11 - x** ... (4)

Dari persamaan (3): **y = 2x - 16** ... (5)

Substitusi (4) dan (5) ke persamaan (1):
```
x + (2x - 16) - (11 - x) = 13
x + 2x - 16 - 11 + x = 13
4x - 27 = 13
4x = 40
x = 10
```

Cari y:
```
y = 2(10) - 16 = 4
```

Cari z:
```
z = 11 - 10 = 1
```

### **Jawaban: x = 10, y = 4, z = 1**

## **f. Sistem Persamaan:**
- 2x + 7y - z = 30
- x - y + z = 6
- 5x - 2y + z = 9

### **Penyelesaian:**

**Langkah 1:** Eliminasi z dari persamaan (1) dan (2)

Tambahkan persamaan (1) dan (2):
```
  2x + 7y - z = 30
+(x - y + z = 6)
─────────────────
  3x + 6y + 0 = 36
```
Hasil: **3x + 6y = 36** atau **x + 2y = 12** ... (4)

**Langkah 2:** Eliminasi z dari persamaan (2) dan (3)

Kurangkan persamaan (3) dari persamaan (2):
```
  x - y + z = 6
-(5x - 2y + z = 9)
──────────────────
  -4x + y + 0 = -3
```
Hasil: **-4x + y = -3** atau **y = 4x - 3** ... (5)

**Langkah 3:** Substitusi (5) ke persamaan (4)
```
x + 2(4x - 3) = 12
x + 8x - 6 = 12
9x = 18
x = 2
```

**Langkah 4:** Cari y
```
y = 4(2) - 3 = 5
```

**Langkah 5:** Substitusi x = 2 dan y = 5 ke persamaan (2)
```
2 - 5 + z = 6
z = 9
```

### **Jawaban: x = 2, y = 5, z = 9**

## **g. Sistem Persamaan:**
- -2x + 3y + 2z = 0
- 2x + y - 6z = 20
- x - 2y - 3z = 0

### **Penyelesaian:**

**Langkah 1:** Eliminasi x dari persamaan (1) dan (2)

Tambahkan persamaan (1) dan (2):
```
  -2x + 3y + 2z = 0
+(2x + y - 6z = 20)
───────────────────
  0 + 4y - 4z = 20
```
Hasil: **4y - 4z = 20** atau **y - z = 5** atau **y = z + 5** ... (4)

**Langkah 2:** Eliminasi x dari persamaan (1) dan (3)

Kalikan persamaan (3) dengan 2:
```
2x - 4y - 6z = 0
```

Tambahkan dengan persamaan (1):
```
  -2x + 3y + 2z = 0
+(2x - 4y - 6z = 0)
───────────────────
  0 - y - 4z = 0
```
Hasil: **-y - 4z = 0** atau **y = -4z** ... (5)

**Langkah 3:** Dari persamaan (4) dan (5)
```
z + 5 = -4z
5z = -5
z = -1
```

**Langkah 4:** Cari y
```
y = -4(-1) = 4
```

**Langkah 5:** Substitusi y = 4 dan z = -1 ke persamaan (3)
```
x - 2(4) - 3(-1) = 0
x - 8 + 3 = 0
x = 5
```

### **Jawaban: x = 5, y = 4, z = -1**

## **i. Sistem Persamaan:**
- -2x + 7y - z = 3
- x - y + 2z = 11
- 7x - 2y + z = 22

### **Penyelesaian:**

**Langkah 1:** Eliminasi x dari persamaan (1) dan (2)

Kalikan persamaan (2) dengan 2:
```
2x - 2y + 4z = 22
```

Tambahkan dengan persamaan (1):
```
  -2x + 7y - z = 3
+(2x - 2y + 4z = 22)
────────────────────
  0 + 5y + 3z = 25
```
Hasil: **5y + 3z = 25** ... (4)

**Langkah 2:** Eliminasi x dari persamaan (2) dan (3)

Kalikan persamaan (2) dengan 7:
```
7x - 7y + 14z = 77
```

Kurangkan dengan persamaan (3):
```
  7x - 7y + 14z = 77
-(7x - 2y + z = 22)
───────────────────
  0 - 5y + 13z = 55
```
Hasil: **-5y + 13z = 55** atau **5y = 13z - 55** ... (5)

**Langkah 3:** Dari persamaan (4) dan (5)
```
13z - 55 + 3z = 25
16z = 80
z = 5
```

**Langkah 4:** Substitusi z = 5 ke persamaan (4)
```
5y + 3(5) = 25
5y + 15 = 25
5y = 10
y = 2
```

**Langkah 5:** Substitusi y = 2 dan z = 5 ke persamaan (2)
```
x - 2 + 2(5) = 11
x - 2 + 10 = 11
x = 3
```

### **Jawaban: x = 3, y = 2, z = 5**

## **j. Sistem Persamaan:**
- 5x + 2y = 19
- 7x - 4z = 1
- 2y + z = 9

### **Penyelesaian:**

Dari persamaan (3): **z = 9 - 2y** ... (4)

Substitusi ke persamaan (2):
```
7x - 4(9 - 2y) = 1
7x - 36 + 8y = 1
7x + 8y = 37
```
... (5)

Dari persamaan (1): **5x + 2y = 19**

Kalikan persamaan (1) dengan 4:
```
20x + 8y = 76
```

Kurangkan dengan persamaan (5):
```
  20x + 8y = 76
-(7x + 8y = 37)
────────────────
  13x + 0 = 39
  x = 3
```

Substitusi x = 3 ke persamaan (1):
```
5(3) + 2y = 19
15 + 2y = 19
2y = 4
y = 2
```

Substitusi y = 2 ke persamaan (3):
```
2(2) + z = 9
4 + z = 9
z = 5
```

### **Jawaban: x = 3, y = 2, z = 5**
