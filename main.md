## **a. Sistem Persamaan:**
- x + 2y + z = 4
- 2x + y + z = 4
- 3x + y - z = 3

### **Penyelesaian:**

**Langkah 1:** Eliminasi z dari persamaan (1) dan (2)
```
  x + 2y + z = 4
-(2x + y + z = 4)
─────────────────
 -x + y + 0 = 0
```
Hasil: **-x + y = 0** atau **y = x** ... (4)

**Langkah 2:** Eliminasi z dari persamaan (2) dan (3)
```
  2x + y + z = 4
+(3x + y - z = 3)
─────────────────
  5x + 2y + 0 = 7
```
Hasil: **5x + 2y = 7** ... (5)

**Langkah 3:** Substitusi y = x ke persamaan (5)
```
5x + 2(x) = 7
5x + 2x = 7
7x = 7
x = 1
```

**Langkah 4:** Cari y
```
y = x = 1
```

**Langkah 5:** Substitusi x = 1 dan y = 1 ke persamaan (1)
```
1 + 2(1) + z = 4
1 + 2 + z = 4
z = 1
```

### **Jawaban: x = 1, y = 1, z = 1**

## **b. Sistem Persamaan:**
- 2x + y + z = 4
- 3x - y + 2z = 4
- x - 2y + 3z = 3

### **Penyelesaian:**

**Langkah 1:** Eliminasi y dari persamaan (1) dan (2)
```
  2x + y + z = 4
+(3x - y + 2z = 4)
──────────────────
  5x + 0 + 3z = 8
```
Hasil: **5x + 3z = 8** ... (4)

**Langkah 2:** Eliminasi y dari persamaan (1) dan (3)

Kalikan persamaan (1) dengan 2:
```
4x + 2y + 2z = 8
```

Tambahkan dengan persamaan (3):
```
  4x + 2y + 2z = 8
+(x - 2y + 3z = 3)
──────────────────
  5x + 0 + 5z = 11
```
Hasil: **5x + 5z = 11** ... (5)

**Langkah 3:** Eliminasi x dari persamaan (4) dan (5)
```
  5x + 3z = 8
-(5x + 5z = 11)
───────────────
  0 - 2z = -3
```
```
-2z = -3
z = 3/2
```

**Langkah 4:** Substitusi z = 3/2 ke persamaan (4)
```
5x + 3(3/2) = 8
5x + 9/2 = 8
5x = 8 - 9/2 = 16/2 - 9/2 = 7/2
x = 7/10
```

**Langkah 5:** Substitusi x = 7/10 dan z = 3/2 ke persamaan (1)
```
2(7/10) + y + 3/2 = 4
7/5 + y + 3/2 = 4
14/10 + y + 15/10 = 4
29/10 + y = 4
y = 4 - 29/10 = 40/10 - 29/10 = 11/10
```

### **Jawaban: x = 7/10, y = 11/10, z = 3/2**

## **d. Sistem Persamaan:**
- x - 8y + 4z = 2
- x - 4y + 7z = 7
- x + 2y - z = 2

### **Penyelesaian:**

**Langkah 1:** Eliminasi x dari persamaan (1) dan (2)
```
  x - 8y + 4z = 2
-(x - 4y + 7z = 7)
──────────────────
  0 - 4y - 3z = -5
```
Hasil: **-4y - 3z = -5** atau **4y + 3z = 5** ... (4)

**Langkah 2:** Eliminasi x dari persamaan (1) dan (3)
```
  x - 8y + 4z = 2
-(x + 2y - z = 2)
──────────────────
  0 - 10y + 5z = 0
```
Hasil: **-10y + 5z = 0** atau **5z = 10y** atau **z = 2y** ... (5)

**Langkah 3:** Substitusi z = 2y ke persamaan (4)
```
4y + 3(2y) = 5
4y + 6y = 5
10y = 5
y = 1/2
```

**Langkah 4:** Cari z
```
z = 2y = 2(1/2) = 1
```

**Langkah 5:** Substitusi y = 1/2 dan z = 1 ke persamaan (3)
```
x + 2(1/2) - 1 = 2
x + 1 - 1 = 2
x = 2
```

### **Jawaban: x = 2, y = 1/2, z = 1**

## **e. Sistem Persamaan:**
- 5x + 2y = 5
- 7x - 4z = -1
- 2y + z = 2

### **Penyelesaian:**

Dari persamaan (3): **z = 2 - 2y** ... (4)

Substitusi ke persamaan (2):
```
7x - 4(2 - 2y) = -1
7x - 8 + 8y = -1
7x + 8y = 7
```
Hasil: **7x + 8y = 7** ... (5)

Dari persamaan (1): **5x + 2y = 5**

Kalikan persamaan (1) dengan 4:
```
20x + 8y = 20
```

Kurangkan dengan persamaan (5):
```
  20x + 8y = 20
-(7x + 8y = 7)
───────────────
  13x + 0 = 13
  x = 1
```

Substitusi x = 1 ke persamaan (1):
```
5(1) + 2y = 5
5 + 2y = 5
2y = 0
y = 0
```

Substitusi y = 0 ke persamaan (3):
```
2(0) + z = 2
z = 2
```

### **Jawaban: x = 1, y = 0, z = 2**

## **f. Sistem Persamaan:**
- -5x + 2y = -7
- 7x - 4y - 4z = -9
- 2y + 2z = 8

### **Penyelesaian:**

Dari persamaan (3):
```
2y + 2z = 8
y + z = 4
z = 4 - y
```
... (4)

Substitusi ke persamaan (2):
```
7x - 4y - 4(4 - y) = -9
7x - 4y - 16 + 4y = -9
7x - 16 = -9
7x = 7
x = 1
```

Dari persamaan (1):
```
-5(1) + 2y = -7
-5 + 2y = -7
2y = -2
y = -1
```

Dari persamaan (4):
```
z = 4 - (-1) = 5
```

### **Jawaban: x = 1, y = -1, z = 5**

## **g. Sistem Persamaan:**
- x + 2y - z = -2
- 2x - 3y - 4z = 1
- x + 2y + 2z = 1

### **Penyelesaian:**

**Langkah 1:** Eliminasi x dari persamaan (1) dan (3)
```
  x + 2y - z = -2
-(x + 2y + 2z = 1)
──────────────────
  0 + 0 - 3z = -3
```
```
-3z = -3
z = 1
```

**Langkah 2:** Substitusi z = 1 ke persamaan (1)
```
x + 2y - 1 = -2
x + 2y = -1
```
... (4)

**Langkah 3:** Substitusi z = 1 ke persamaan (2)
```
2x - 3y - 4(1) = 1
2x - 3y - 4 = 1
2x - 3y = 5
```
... (5)

**Langkah 4:** Eliminasi x dari persamaan (4) dan (5)

Kalikan persamaan (4) dengan 2:
```
2x + 4y = -2
```

Kurangkan dengan persamaan (5):
```
  2x + 4y = -2
-(2x - 3y = 5)
───────────────
  0 + 7y = -7
  y = -1
```

**Langkah 5:** Substitusi y = -1 ke persamaan (4)
```
x + 2(-1) = -1
x - 2 = -1
x = 1
```

### **Jawaban: x = 1, y = -1, z = 1**

## **h. Sistem Persamaan:**
- 6x + 2y - z = 3
- 8x - 2y - z = 9
- 3x + 2y + z = 2

### **Penyelesaian:**

**Langkah 1:** Eliminasi y dari persamaan (1) dan (2)
```
  6x + 2y - z = 3
+(8x - 2y - z = 9)
──────────────────
  14x + 0 - 2z = 12
```
Hasil: **14x - 2z = 12** atau **7x - z = 6** ... (4)

**Langkah 2:** Eliminasi y dari persamaan (1) dan (3)

Kalikan persamaan (1) dengan -1:
```
-6x - 2y + z = -3
```

Tambahkan dengan persamaan (3):
```
  -6x - 2y + z = -3
+(3x + 2y + z = 2)
───────────────────
  -3x + 0 + 2z = -1
```
Hasil: **-3x + 2z = -1** ... (5)

**Langkah 3:** Eliminasi z dari persamaan (4) dan (5)

Dari persamaan (4): z = 7x - 6

Substitusi ke persamaan (5):
```
-3x + 2(7x - 6) = -1
-3x + 14x - 12 = -1
11x = 11
x = 1
```

**Langkah 4:** Cari z
```
z = 7(1) - 6 = 1
```

**Langkah 5:** Substitusi x = 1 dan z = 1 ke persamaan (3)
```
3(1) + 2y + 1 = 2
3 + 2y + 1 = 2
2y = -2
y = -1
```

### **Jawaban: x = 1, y = -1, z = 1**
